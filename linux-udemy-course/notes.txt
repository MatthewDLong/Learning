Useful commands:

Overview of system:
glances

Manage processes:

sudo kill -HUP PID # reload
sudo kill -TERM PID # gracefully stop
sudo kill -KILL PID # force stop

Get the status of nginx:
systemctl status nginx

Check who is logged in and what command did they run:
w

history

du -h --max-depth=1 *

ps -ef # All processes in full format

get a pid e.g. 704

sudo ls /proc/704/fd # See file descriptors for given process

apt commands:

apt list glances

apt list --installed

sudo apt-get install

sudo apt remove glances

apt-get install glances

Sorting:

users.csv content:
id,name,count
0,foo,1
2,baz,3
1,bar,5

sort -t , -k1,1 < users.csv # sort with comma delimited fields, numerical sort by column 1 (id)
output:
0,foo,1
1,bar,5
2,baz,3
id,name,count

sort -t , -k2,2 -d < users.csv # sort with comma delimited fields, dictionary sort by column 2 (name)
output:
1,bar,5
2,baz,3
0,foo,1
id,name,count

sort -t , -k2,2 -d -r < users.csv # reverse order, sort with comma delimited fields, dictionary sort by column 2 (name)

nmap localhost

Install net tools:

List route table:

Show IP addresses:

Show network interfaces:
ip link

Bring down interface:
ip link set eth0 down

Bring up interface:
ip link set eth0 up
